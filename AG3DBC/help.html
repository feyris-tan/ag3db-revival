If you encounter any strange errors please send me the error log OF THAT PARTICULAR ERROR, which can be found in the 'Logs' folder of your AG3DBC installation folder to errors@ag3.illusionist.sosavalanche.net with subject AG3DBC Error and a brief description of how the error occured.

Search:
  • Wilds:
    • With 'Match Any Field' selected: Wild fields count as a search miss.
    • With 'Match All Fields' selected: Wild fields count as a search hit.
  • Ranges are inclusive
BBCode:
  • Each tag can be modified through the cascading style sheet 'styles.css', which can be found in the 'Tools\bbcode' folder
Options:
  • Right-click on either the 'Make' or 'Play' buttons to change their exe locations respectively.
  • Click on either of the color squares in the 'Color' options tab to change colors.
Status Bar:
  • Click on the message text to open the current log file.
  • Right-click on the message text to open the current errors log file.
Other important information to note:
  • The program assumes that the registry values for AG3 have been unmodified and that all folders inside the AG3 'data' folder have not been moved or renamed.
  • The program needs to access the AG3DB web server so if you have a software firewall running you'll probably need to add AG3DBC to its list of exceptions or else you probably wont be able to log into the database.
